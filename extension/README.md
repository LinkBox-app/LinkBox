# LinkBox 浏览器插件

该目录提供了一个用于 LinkBox 项目的浏览器插件（基于 Chrome/Edge Manifest V3）。插件可以从当前标签页快速生成网页摘要并保存到自建的 LinkBox 服务中。

## 功能概览

- 登录或注册 LinkBox 账户，并在浏览器中保存认证信息。
- 一键读取当前标签页的标题和链接。
- 调用服务端 `/resources/preview` 接口生成 AI 摘要与标签建议。
- 手动编辑标题、摘要与标签后调用 `/resources/` 接口完成收藏。
- 在弹窗中刷新已有标签列表，快速点选常用标签。
- 在选项页配置 LinkBox 服务端地址，并测试健康检查接口。
- 一键清除本地存储的登录凭证。

## 本地加载步骤

1. 构建并启动 LinkBox 的后端与前端服务（默认后端地址为 `http://localhost:7032`）。
2. 打开 Chromium 内核浏览器（Chrome、Edge、Brave 等）的扩展管理页：
   - Chrome：访问 `chrome://extensions/`
   - Edge：访问 `edge://extensions/`
3. 开启「开发者模式」。
4. 点击「加载已解压的扩展程序」，选择本项目的 `extension/` 目录。

加载完成后，可在浏览器工具栏看到「LinkBox 收藏器」图标。

## 使用说明

1. **首次使用：**
   - 点击插件图标，弹出窗口中输入后端账户的用户名与密码完成登录或注册。
   - 如需连接到非默认地址，在弹窗底部点击「选项」，或在扩展详情页打开「扩展程序选项」，填写新的服务端地址并保存。
2. **生成预览：**
   - 确认浏览器当前标签页为需要收藏的网页，点击「生成预览」。
   - 插件会调用后台接口获取标题、摘要和标签建议，可自行调整。
3. **保存收藏：**
   - 至少保证标题、摘要和一个标签不为空。
   - 点击「保存到 LinkBox」即可完成收藏。
4. **标签管理：**
   - 「我的标签」折叠面板展示服务器中已有的标签，点击即可添加。
   - 支持手动输入新标签，按回车或点击「添加」按钮即可加入。
5. **退出登录：**
   - 在弹窗底部点击「退出登录」可移除本地凭证。
   - 也可在选项页点击「清除登录信息」。

## 注意事项

- 插件默认允许访问所有网址（`"<all_urls>"`），以便连接任意部署的 LinkBox 服务；如需限制，可在 `manifest.json` 中收紧 `host_permissions`。
- 服务器需开启 CORS（项目默认允许全部源），否则插件无法成功请求 API。
- 生成预览及保存功能依赖 `/resources/preview` 与 `/resources/` 接口；如后端配置或鉴权发生改变，请同步调整插件逻辑。

如需进一步定制插件功能（例如添加右键快捷菜单、内容脚本等），可基于现有结构继续扩展。
